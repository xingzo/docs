## About the Download Library

The download library stores your recently downloaded tracks for 24 hours after initial download.

### Key Features

- 24-hour access to recent downloads
- No additional credit usage for redownloads
- Easy access to all track variants
- Automatic cleanup after 24 hours

<Tip>
Check the library first if downloads fail - you can redownload without using additional credits.
</Tip>
