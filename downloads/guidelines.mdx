## Best Practices

- Don't use download managers or browser add-ons
- Allow downloads to complete naturally
- If a download fails, don't resume - start fresh
- Use the download library for recent downloads

<Warning>
Using download managers or browser add-ons can block or duplicate downloads.
</Warning>
